# ECRL-acad-tree

Academic tree centered around Prof. Mack. To view the tree as a visual graph, click on "acad_tree.md" or "acad_tree_LR.md" in the list of files here on Github.

The tree has been created as a Mermaid chart in a ".md" file, which should render automatically on Github. It will also open in software like Obsidian and VS Code that render Mermaid in .md markdown files.
